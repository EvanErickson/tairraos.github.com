安装小乐的 TamperMonkey 自制脚本指南

  • 点击安装油猴插件 要继续下面操作前，请确定你已经成功安装插件。

安装 WebEx UT 覆盖率比较工具

  • 安装：点击安装Webex UT Tool 在新窗口点击install
  • 使用：访问 jenkins系统 使用工具
  • 维护：不需要维护，自制脚本如果有bug，小乐修复后会自动升级

安装 Get session ticket工具

  • 安装：点击安装Webex Session Ticket获取工具 在新窗口点击install
  • 使用：qa.webex.com / qa.webex.com.cn 的站点，登录后来到启会/加会页面会在页面上显示session ticket
  • 维护：如果需要关闭功能，在加会页面，点击插件图标，把相应脚本开关关闭
